## 2. /llms.txt

**PASS.** Served live as `content-type: text/plain; charset=utf-8`. The file is
**generated at every build** by `scripts/build-llms.mjs` from `src/data/company.mjs`
(the same module CLAUDE.md's canonical facts mirror), so prices ($3,500–$8,500,
$12,000–$45,000), phone (480-360-5128), cities, ~90 days, and the free assessment
can never drift — hand-edits are overwritten at the next build.

## 3. sitemap.xml

**PASS (after FIX).** 24 URLs in the local build (22 static routes + /blog/ +
/blog/archive/; essay URLs join at deploy when beehiiv content is fetched). Audit
confirmed: every URL exactly once, all canonical trailing-slash form, https + www
host only, zero fragments. `robots.txt` references the sitemap and disallows only
`/api/`. **FIX applied this round:** `/blog`, `/blog/archive`, and essay URLs
(`fetch-blog-posts.mjs` built them slash-less) were standardized to trailing-slash
form across hrefs, canonicals, JSON-LD, RSS, and the sitemap generator.

## 4. Essay URLs

**PASS.** Every essay renders at a first-class on-site URL (`/blog/<slug>/` —
verified live, 8 spot-checked URLs all 200), each with a unique title
(`<Essay Title> — The Ampersand | Main & Machine`) and per-post meta description
(beehiiv `seoDescription` falling back to the excerpt). Posts enter the sitemap at
build. `seo:check` fails the deploy if a post page is missing its canonical,
single H1, or BlogPosting JSON-LD.

## 5. Canonicals

**PASS.** All 22 static pages carry a self-referencing canonical on the
`https://www.` host in trailing-slash form (machine-verified, zero mismatches);
blog pages are generated with the same convention. The apex 301s to www at the
Cloudflare edge per robots.txt; Cloudflare Pages 308s slash-less directory paths
to the canonical slashed form, and after this round **no internal link resolves
through that redirect**.
